# app

## Build Setup

````bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

#Migrating

- Chnage 'rouser.base in config'
- Chnage '/nuxt/shariah-compliant-financing' in pages/index.vue
- Change '/nuxt/shariah-compliant-financing' in components/Header
- Change '/nuxt/media' in components/Header
- Change '/nuxt/sahih/media' in components/HeaderSahih
- Change '/nuxt/' in components/HeaderSahih
- Change API_KEY in .env, if neccessary
- Change GA tag in 'plugins/ga.js'

````
